\doxysubsection{Detailed Description}
Uses Open\+X\+ML to perform a Word Mail Merge. 

It\textquotesingle{}s quick, but no always very reliable. Perhaps this class needs some modifications... Also, Open\+X\+ML doesn\textquotesingle{}t have the facility to convert to P\+DF, so we have to use the M-\/\+Files A\+PI to do this for us. It\textquotesingle{}s usually quick, but can be slow if M-\/\+Files is busy.
