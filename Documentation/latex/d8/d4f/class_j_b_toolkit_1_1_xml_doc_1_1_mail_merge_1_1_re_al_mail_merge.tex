\doxysubsection{Detailed Description}
Mail merging using a 3rd party command line tool. 

It\textquotesingle{}s generally quick and reliable, however it must load the proces which then loads its assemblies. An implementation using .Net managed code libraries would be more effecient. Especially when performing mail merges in bulk.
