Detailed Description

An ASCII progress bar

Use like:

Console.Write("Performing some task... "); using (var progress = new ProgressBar()) { for (int i = 0; i <= 100; i++) { progress.Report((double) i / 100); Thread.Sleep(500); } } Console.WriteLine("Done.");
